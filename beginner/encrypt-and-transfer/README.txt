ALL DEPENDENCIES WERE INSTALLED ONTO THE VMS BEFORE THE PROJECT WAS STARTED

This project is to create an encryption key and send encrypted data from one VM to another (Kali - Windows) and then have the other VM de-crypt the data. This project showcased the ability to connect one OS to another, send files encrypted files, and with the same key de-crypt those encrypted files (Python).

Relevant Data:

eat-information | Setup and general information of the process.
eat-screenshots | Important screenshots through the encrypt-and-decrypt process.
decrypt.py | The decryption logic.
encrypt.py | The encryption logic.
sample-output | Where the key and encrypted key will be held.
